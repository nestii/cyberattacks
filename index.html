IoT - Mirai

  • Mirai is a malware which infects IoT devices. It uses them as a platform for further DDoS attacks.
  • It's coded in GO, while the bots are coded in C
  • It builds a botnet

How does Mirai work?

  • Mirai performs a wide range scan of IPs to locate not (or weakly) secured IoT devices.
  • It uses dictionary attack to guess credentials
  • It builds a botnet
  • There was a hardcoded list in code with IPs, which schould be avoided during scan (such as the US Postal Service, the Department of Defense etc.)
  • It contains some scripts which are meant to kill other worms and trojans and competitors such as Anime malware.
